# Golf Charity Platform

## Description
This project is a web application where users can log in, submit their golf scores, and participate in a monthly draw.

## Features
- User login interface
- Score submission system
- Only last 5 scores are stored
- Admin panel for random draw

## Technologies Used
- Next.js
- React

## Working
Users enter their scores in the dashboard. The system keeps only the latest 5 scores. The admin can run a random draw which generates random winning numbers.
